Hello World!

In most tutorials the first program prints out the words "Hello World". Below is a simple program printing out "Hello World! This is my first program". At this time you don't need to concern yourself with the strange Java code lines. 

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!  This is my first program.");   // Note that this is an important paragraph. :) 
    }
}

line 2: Ignore the public static void main.

line 3: System.out.println allows something to printed on the console.
